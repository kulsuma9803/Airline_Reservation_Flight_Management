# Airline Reservation & Flight Management System (Console Based)
A Java console-based Airline Reservation System developed using complete Object-Oriented Programming principles.
This system manages flight operations, ticket booking, cancellation, admin controls, and report generation efficiently.
## Key Features
- View and Search Flights
- Book Ticket
- Cancel Ticket
- Admin Panel Management
- View All Bookings
- Generate System Reports
- Background Auto Backup using Daemon Thread
## OOP Concepts Implemented
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Classes & Objects
- Method Overloading / Overriding
- Constructor Usage
## Additional Concepts
- Multithreading (Daemon Thread)
- Exception Handling
- Collections Framework (if used)
- Modular Code Structure
## Technologies Used
- Java
- NetBeans IDE
## How to Run
Compile
-javac AirlineSystem.java
Run:
-java AirlineSystem
